SIDEBARS COLAPSÁVEIS:
================================================================

- Árvore de Arquivos (esquerda) - COLAPSADA por padrão
  * Clique em "Ver > Árvore de Arquivos" para abrir/fechar
  * Ou use Ctrl+B

- Snippets (direita) - COLAPSADA por padrão
  * Clique em "Ver > Snippets" para abrir/fechar

================================================================
INTEGRAÇÃO COM SISTEMA DE ARQUIVOS:
================================================================

Todas as opções do menu "Arquivo" estão integradas:
  ✓ Novo Arquivo → Cria arquivo no projeto
  ✓ Nova Pasta → Cria pasta no projeto
  ✓ Upload → Envia arquivos para o projeto
  ✓ Download ZIP → Baixa projeto completo

================================================================
COMO USAR:
================================================================

1. Clique em qualquer item do menu
2. O dropdown abre automaticamente
3. Clique na opção desejada
4. Use os atalhos de teclado para rapidez

EXEMPLO:
  - Clique em "Arquivo"
  - Clique em "Novo Arquivo"
  - Digite o nome (ex: capitulo1.tex)
  - Pronto! Arquivo criado no projeto
