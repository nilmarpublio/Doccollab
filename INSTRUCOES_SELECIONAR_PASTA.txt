TROUBLESHOOTING:
================================================================

PROBLEMA 1: Botão 📂 não aparece
SOLUÇÃO: Recarregue a página (F5)

PROBLEMA 2: Janela do explorador não abre
SOLUÇÃO: Verifique se está usando Chrome ou Edge
         Atualize o navegador

PROBLEMA 3: "Seu navegador não suporta..."
SOLUÇÃO: Use Chrome 86+ ou Edge 86+
         Atualize o navegador

PROBLEMA 4: Selecionei mas não aparece nada
SOLUÇÃO: Verifique se selecionou a pasta correta:
         D:\OurDocs\DocCollab\Arquivos
         (não uma subpasta!)

PROBLEMA 5: "Erro ao selecionar pasta"
SOLUÇÃO: Clique em "Permitir" quando solicitado
         Verifique permissões do navegador

PROBLEMA 6: Arquivos desaparecem ao recarregar
SOLUÇÃO: Normal! Permissões são temporárias
         Clique em 📂 novamente

================================================================
CONSOLE DO NAVEGADOR (DEBUG):
================================================================

Se ainda não funcionar, abra o console:

1. Pressione F12
2. Vá para a aba "Console"
3. Clique no botão 📂
4. Veja se há erros em vermelho

Erros comuns:
- "showDirectoryPicker is not defined"
  → Use Chrome ou Edge

- "User cancelled the picker"
  → Você cancelou, tente novamente

- "Permission denied"
  → Clique em "Permitir"

================================================================
ALTERNATIVA: BOTÕES NOVO/ABRIR/SALVAR
================================================================

Se o botão 📂 não funcionar, você também pode:

1. NOVO DOCUMENTO:
   - Clique em "Novo" na toolbar
   - Digite o nome
   - Ctrl+S para salvar
   - Selecione a pasta "Arquivos"

2. ABRIR DOCUMENTO:
   - Clique em "Abrir" na toolbar
   - Navegue até a pasta "Arquivos"
   - Selecione um arquivo

3. SALVAR DOCUMENTO:
   - Ctrl+S
   - Selecione a pasta "Arquivos"

Qualquer uma dessas ações também solicitará
a seleção da pasta!
