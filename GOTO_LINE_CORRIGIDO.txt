🧪 COMO TESTAR:
════════════════════════════════════════════════════════════════

1. Abra o editor LaTeX
2. Cole um documento com erros (ex: TESTE_ERROS_BASICOS.txt)
3. Clique no botão "LaTeX Linter"
4. Clique em "Ir para linha" em qualquer erro
5. O editor irá:
   ✅ Mover o cursor para a linha
   ✅ Fazer scroll para mostrar a linha
   ✅ Destacar a linha por 1 segundo
   ✅ Mostrar notificação de sucesso
   ✅ Fechar o painel do linter
