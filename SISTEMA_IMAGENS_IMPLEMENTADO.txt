COMO USAR:
================================================================

1. FAZER UPLOAD DE IMAGEM:
   
   Opção 1: Botão "Imagem" na toolbar
   - Clique no botão verde "Imagem"
   - Selecione uma ou várias imagens
   - Confirme
   
   Opção 2: Botão na seção "Imagens"
   - Clique no ícone de upload (↑)
   - Selecione imagens
   
   Opção 3: Drag-and-drop
   - Arraste imagens para a seção "Imagens"
   - OU arraste diretamente no editor

2. INSERIR IMAGEM NO DOCUMENTO:
   
   - Clique no botão "Inserir" da imagem
   - Código LaTeX será inserido automaticamente
   - Edite a legenda (já selecionada)
   - Ajuste o tamanho se necessário

3. EXCLUIR IMAGEM:
   
   - Passe o mouse sobre a imagem
   - Clique no botão vermelho "Excluir"
   - Confirme a exclusão

================================================================
CÓDIGO LATEX GERADO:
================================================================

Ao clicar em "Inserir", o seguinte código é gerado:

\begin{figure}[h]
  \centering
  \includegraphics[width=0.8\textwidth]{imagens/minha_imagem.png}
  \caption{Legenda da imagem}
  \label{fig:minha_imagem}
\end{figure}

PARÂMETROS AJUSTÁVEIS:

- width=0.8\textwidth  → Largura (80% do texto)
- [h]                  → Posição (here)
- \caption{...}        → Legenda
- \label{fig:...}      → Referência

OUTRAS OPÇÕES:

% Imagem simples (sem figure)
\includegraphics[width=5cm]{imagens/logo.png}

% Imagem com altura fixa
\includegraphics[height=8cm]{imagens/grafico.jpg}

% Imagem rotacionada
\includegraphics[angle=90,width=0.5\textwidth]{imagens/diagrama.pdf}

% Duas imagens lado a lado
\begin{figure}[h]
  \centering
  \begin{minipage}{0.45\textwidth}
    \includegraphics[width=\textwidth]{imagens/img1.png}
    \caption{Primeira imagem}
  \end{minipage}
  \hfill
  \begin{minipage}{0.45\textwidth}
    \includegraphics[width=\textwidth]{imagens/img2.png}
    \caption{Segunda imagem}
  \end{minipage}
\end{figure}

PACOTES LATEX NECESSÁRIOS:
================================================================

Para usar imagens no LaTeX, adicione ao preâmbulo:

\usepackage{graphicx}  % Obrigatório para \includegraphics
\usepackage{float}     % Para controle de posicionamento [H]
\usepackage{caption}   % Para legendas personalizadas
\usepackage{subcaption}% Para subfiguras

Exemplo de preâmbulo completo:

\documentclass[12pt,a4paper]{article}
\usepackage[utf8]{inputenc}
\usepackage[portuguese]{babel}
\usepackage{graphicx}
\usepackage{float}
\usepackage{caption}

\graphicspath{{imagens/}} % Define pasta padrão

\begin{document}
...
\end{document}
